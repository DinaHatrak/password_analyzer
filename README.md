# Password Security Analyzer

## Description:
A comprehensive Python tool designed to evaluate password strength using mathematical entropy and NIST standards. This project was developed as part of the CS50 curriculum to demonstrate secure coding practices and data analysis.

## Project Structure:
- **project.py**: The main entry point of the application.
- **analysis/**: Contains logic for Shannon Entropy and Brute Force estimation.
- **src/**: Includes NIST compliance checks and security utilities.
- **tests/**: Unit tests to ensure code reliability.

## How to Run:
Run the main script using:
`python project.py`
